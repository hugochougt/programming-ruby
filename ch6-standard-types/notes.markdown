## Number

数值较小的 number 是 *Fixnum* 类型，数值较大的 number 是 *Bignum* 类型。`

从文件读入的数字字符串不会自动转换成 number 类型。

引入 *mathn* 库会使整型除法返回分数值（a Rational number）。

数值类型还自带很多方法，例如 times, upto, downto, step 等。

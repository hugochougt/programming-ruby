In Ruby, just about everything is an expression. So perhaps less obvious, things that are normally statements in C or Java are expresisons in Ruby.

Ruby **没有** `++` 和 `--` 两个操作符。

## Boolean Expressions

Any value that is not `nil` or the constant `false` is *true*.
